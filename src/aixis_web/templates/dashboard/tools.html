認証Cookie設定

ログインが必要なツールの場合、以下の手順でCookieを設定してください。

Cookieの取得手順（方法1: 推奨）

  1. 対象ツールのサイトにアクセスし、ログインする
  2. ログイン後、F12キー（Mac: Cmd+Option+I）でDevToolsを開く
  3. Application タブ → 左の Cookies → 対象ドメインをクリック
  4. Cookie一覧が表示される → 表の任意のセルをクリック → Ctrl+A（全選択）
  5. 右クリック → 「Copy」を選択（全行がコピーされます）
  6. 下の入力欄に Ctrl+V / Cmd+V で貼り付けて「保存」

※ この方法はHttpOnly Cookieも含むため最も確実です

方法2: Consoleコマンド（HttpOnly Cookieは含まれません）

※ この方法ではHttpOnly Cookieが取得できないため、ログインが維持されない場合があります

  1. DevTools → Console タブを開く
  2. 初回は「allow pasting」と入力してEnter
  3. 下のコマンドを貼り付けてEnter
  4. Enter後3秒以内にページ側をクリック
setTimeout(()=>cookieStore.getAll().then(c=>{const f=c.map(x=>({name:x.name,value:x.value,domain:x.domain||location.hostname,path:x.path||'/'}));navigator.clipboard.writeText(JSON.stringify(f)).then(()=>alert(f.length+'件コピー完了！貼り付けてください'))}),3000)
