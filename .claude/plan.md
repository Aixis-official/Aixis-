# Aixis プラットフォーム改修プラン（第2フェーズ）

## 概要
プロフェッショナルなB2B SaaS品質への改修。無料/有料の境界明確化、料金体系整理、メール送信実装、管理者アクセス整備。

---

## Task 1: ランディングページ料金セクション改修（landing.html）

**変更内容:**
- 現在の3プラン（スポット/ベンチマーク/ガバナンス）を削除
- 新しい2本立て構成に変更:
  - **プラットフォーム利用**: 月額 ¥29,800（税別）— 全監査レポートDB閲覧・比較・検索
  - **個別監査レポート**: 1件 ¥29,800〜 — 単体購入も可能。詳細はお問い合わせ
- CTAは全て `/platform/contact` に誘導

**5軸監査モデル図の追加:**
- `/verification` ページのレーダーチャート風のSVG/CSS図をランディングページに実装
- 5軸（実務適性・費用対効果・日本語能力・信頼性・安全性・革新性）を五角形レーダー風SVGで表現
- ダークモード対応、レスポンシブ対応

## Task 2: ツール一覧の無料/有料境界（tools.html）

**変更内容:**
- ツールカードに表示する情報を制限:
  - **公開**: ツール名、ベンダー、カテゴリ、総合グレード（S/A/B/C/D バッジ）
  - **非公開（ぼかし）**: 5軸個別スコアバー → `filter: blur(6px)` + オーバーレイ
- 各カードの5軸スコアバー部分に半透明オーバーレイ:
  - テキスト: 「詳細スコアは有料プランでご確認いただけます」
  - 小さな鍵アイコン（SVG）
- ページ上部にも注釈: 「総合グレードは無料でご覧いただけます。5軸の詳細スコア・分析レポートは有料プランをご利用ください。」

## Task 3: ツール詳細ページの無料/有料境界（tool_detail.html）

**変更内容:**
- **公開セクション**: ツール基本情報、総合グレード・総合スコア、簡易サマリー（強み/弱みの概要テキスト）
- **有料ゲート**: 5軸レーダーチャート、5軸スコアテーブル、詳細レポート → 全てblur + ペイウォールオーバーレイ
- ペイウォールCTA: 「月額¥29,800で全データベースにアクセス」+「お問い合わせ」ボタン
- レーダーチャートのスタイルをランディングページのSVG図と統一感のあるカラーリングに

## Task 4: お問い合わせフォーム SMTP実装（contact.py）

**変更内容:**
- `config.py` に SMTP設定を追加:
  ```
  smtp_host: str = ""
  smtp_port: int = 587
  smtp_user: str = ""
  smtp_password: str = ""
  smtp_from: str = "noreply@aixis.jp"
  smtp_to: str = "info@aixis.jp"
  ```
- `contact.py` を改修:
  - SMTP設定がある場合 → `smtplib` でメール送信（`aiosmtplib` は依存追加が必要なので標準ライブラリの `smtplib` を `run_in_executor` で非同期化）
  - SMTP未設定の場合 → ログ記録のみ（現状維持、開発環境用）
  - メール本文: 会社名、部署、氏名、メール、電話、お問い合わせ種別、本文を整形
- `.env.example` にSMTP設定例を記載
